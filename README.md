# 🍔 SwiftEats — Premium Food Delivery Website

SwiftEats is a modern and responsive food delivery landing page built using HTML, CSS, Bootstrap 5, JavaScript, AOS Animations, and Font Awesome.

This project is designed with Fiverr premium-level UI/UX styling and includes smooth animations, interactive sections, responsive layout, and modern food ordering design.

---

# 🚀 Features

✅ Fully Responsive Design  
✅ Premium Modern UI  
✅ Bootstrap 5 Layout  
✅ Animated Hero Section  
✅ Interactive Food Cards  
✅ Smooth Scroll Navigation  
✅ FAQ Accordion  
✅ Toast Notifications  
✅ Animated Loader  
✅ Back To Top Button  
✅ AOS Scroll Animations  
✅ Mobile Friendly  
✅ Clean & Organized Code
